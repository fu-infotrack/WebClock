# WebClock

A full-screen digital clock web page designed for iPad Safari on iOS 12.5.8.

## Features

- **Full-screen digital clock** with large, centered time display
- **12-hour format** with AM/PM indicator (smaller size, positioned separately)
- **Current date display** showing weekday, month, day, and year
- **Black background** with white text
- **Screen sleep prevention** compatible with iOS 12.5.8 using silent video technique
- **Responsive design** adapts to portrait and landscape orientations
- **No seconds** - displays hours and minutes only
- **Time occupies 90% width** for maximum visibility

## Compatibility

- Optimized for iPad Safari on iOS 12.5.8
- Uses silent video method to prevent screen sleep on older iOS versions
- Tap the screen to activate sleep prevention

## Usage

Open the page in Safari and tap anywhere on the screen to activate screen sleep prevention. The status indicator will confirm when active.
